## AISSE Development Workflow Suggestion


This workflow is provides developers who wish to use the framework and feature in the directory a set of straightforward steps to use. It's arranged to align with the typical activities you'd perform whether you're developing a concept, prototype or release version of your system. These steps run alongside your regular development work, but also inform and reflect on functionality development.


Links to templates on the framework platform are provided, but you need to clone the template directory to your own repository and fill these out there.


Icons in the first column refer to AISSE Directory "milestones" that will be visible on the dashboard
